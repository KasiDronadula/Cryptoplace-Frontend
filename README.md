# 💰 CryptoPlace - Crypto Marketplace (Frontend)

CryptoPlace is a responsive web application that allows users to explore, search, and track cryptocurrencies in real time.  
It provides live market data, price updates, and a smooth user experience for crypto enthusiasts.

---

## 🚀 Features

- 🔍 Search cryptocurrencies instantly  
- 💱 Change currency (USD / EUR / INR)  
- 📊 Live crypto prices and market data  
- 📈 Price charts visualization  
- 🔐 User authentication (Login & Register)  
- 📱 Fully responsive design  
- ⚡ Fast API integration  

---

## 🛠 Technologies Used

- ⚛️ React.js  
- 🎨 CSS3  
- 🔗 Axios / Fetch API  
- 🌐 React Router  
- 📊 Chart.js / Recharts
